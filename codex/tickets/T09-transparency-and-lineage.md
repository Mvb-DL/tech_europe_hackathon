# T09 — Transparency and Lineage

**Status:** TODO  
**Human owners:** Robin and Mario  
**Depends on:** T04–T08

## Goal

Make every visible object and action understandable.

## Scope

- consistent status vocabulary
- source and lineage inspector
- activity stream filtering by layer and subject
- `Why is this here?` explanation
- `Trace source` navigation
- clear demo, candidate, observed, and pending labels
- missing-data states
- processing errors with recovery actions

## Acceptance criteria

- Robin can explain every map object from the UI alone
- user can trace a profile back to at least one uploaded file
- simulated content is never presented as verified evidence
- map stays focused and does not become a log dashboard

## Out of scope

- proof bundles
- findings
- audit conclusions
